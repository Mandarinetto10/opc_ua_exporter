# Certificates Directory

This folder contains certificates generated by the [Generate Certificate Command](../README.md#generate-certificate-command) of OPC UA Exporter.

- Certificates are required for secure connections and are created by running the `generate-cert` command.
- For details and usage, see the [Generate Certificate Command section](../README.md#generate-certificate-command) in the main README.

_This directory is auto-created if it does not exist._
