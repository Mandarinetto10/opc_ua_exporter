# Export Directory

This folder contains files generated by the [Export Command](../README.md#export-command) of OPC UA Exporter.

- Each export file (CSV, JSON, XML) is created when you run the `export` command.
- For usage instructions and export options, see the [Export Command section](../README.md#export-command) in the main README.

_This directory is auto-created if it does not exist._
